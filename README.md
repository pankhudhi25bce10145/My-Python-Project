*INTRODUCTION

Body Mass Index(BMI) is a measure of body fat based on weight and height. It is a tool that estimates a healthy weight for a person's height. It is calculated by dividing a person's weight in kilograms by the square of their height in meters. While it's a useful screening tool, BMI dosesn't account for muscle mass, so it should be used alongside other health assessment.

                                                        BMI= Weight(kg)/Height(m)**2
